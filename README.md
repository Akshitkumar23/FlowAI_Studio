# FlowAI Studio

Welcome to FlowAI Studio, a powerful and innovative platform built with Next.js that provides a suite of AI-powered tools to enhance your creative workflow.

## Features

FlowAI Studio offers a variety of tools to help you generate and refine content:

-   **Creative Spark:** Get new ideas and inspiration for your projects.
-   **PPT Enhancer:** Automatically improve your PowerPoint presentations.
-   **Presentation Generator:** Create complete presentations from a simple prompt.
-   **Prompt Enhancer:** Refine your prompts to get better results from AI models.
-   **Scene Preview:** Visualize scenes for your videos or stories.
-   **Text to Speech:** Convert text into natural-sounding audio.

## Getting Started

To get started with the project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Akshitkumar23/FlowAI_Studio.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd FlowAI_Studio
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Run the development server:**
    ```bash
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
